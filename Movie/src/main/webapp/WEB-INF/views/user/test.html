우대적용

• 국가유공자  현장에서 국가유공자증을 소지한 본인 외 동반 1인까지 적용
• 장애인  현장에서 복지카드를 소지한 장애인(1급~3급까지 동반 1인까지 적용 / 4급~6급은 본인에 한함)
• 경로  만65세 이상 본인에 한함(신분증 확인)
• 미취학아동  부모 동반한 만4세~ 만6세까지 본인(의료보험증, 주민등록 초/등본 확인)
• 소방종사자  공무원증을 소지하거나 정복을 입은 소방관 본인
